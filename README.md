# yuqingc.github.io

- Clone & update submodules (themes)

    ```
    $ git clone --recursive git@github.com:yuqingc/yuqingc.github.io.git
    ```
- Write new post

    ```
    $ hugo new posts/article.md
    ```

- Start

    ```
    $ hugo server -D
    ```
    
- Build

    ```
    $ hugo -d .
    ```
